This folder includes the interactions with Bayesian Factor (BF) scores that calculated by using out series NB models. 
The negative number means inter-chromosomal interactions.
The interactions are formatted as Pairwise interaction format (http://wiki.wubrowse.org/Long-range), which can be uploaded for
to be direct visualization through WashU Genome browser http://epigenomegateway.wustl.edu/legacy/.
